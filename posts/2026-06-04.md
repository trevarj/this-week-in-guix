# This Week in Guix: 2026-06-04
<!-- date: 2026-06-04 -->

This draft covers public Guix activity collected from 2026-05-28 through 2026-06-04.

Collection note: package changes are sampled from a capped diff.

## Top Stories

### Change-Id Trailer Discussion

The highest-scoring thread in this bundle was the guix-devel discussion about whether Guix should remove the Change-Id commit hook and trailer: [Consensus on removal of Change-Id commit-msg hook/git trailer?](https://lists.gnu.org/archive/html/guix-devel/?q=Consensus%20on%20removal%20of%20Change-Id%20commit-msg%20hook/git%20trailer%3F). A related thread discussed adopting Git trailers for issue references: [Adopting 'Fixes' and 'Merges' git trailers](https://lists.gnu.org/archive/html/guix-devel/?q=Adopting%20%27Fixes%27%20and%20%27Merges%27%20git%20trailers).

### GCD 008 Drew Follow-Up

GCD 008, "Standing up for human crafting", drew sustained discussion across related guix-devel threads. Start with [Submitting GCD 008: "Standing up for human crafting"](https://lists.gnu.org/archive/html/guix-devel/?q=Submitting%20GCD%C2%A0008%3A%20%E2%80%9CStanding%20up%20for%20human%20crafting%E2%80%9D), with a related follow-up collected as [Submitting GCD 008: "Standing up for humancrafting"](https://lists.gnu.org/archive/html/guix-devel/?q=Submitting%20GCD%20008%3A%20%E2%80%9CStanding%20up%20for%20humancrafting%E2%80%9D).

### AGit Workflow Discussion

Contributor workflow remained an active topic after the Codeberg migration, with the [AGit workflow](https://lists.gnu.org/archive/html/guix-devel/?q=AGit%20workflow) thread scoring near the top of the guix-devel activity collected this week.

### Branch Merge Requests

Several team or rebuild branches surfaced on guix-patches. The largest was [[bug#80782] Request for merging "misc-world-rebuild" branch](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2380782%5D%20Request%20for%20merging%20%22misc-world-rebuild%22%20branch), with additional merge-request threads for the [go-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2380936%5D%20Request%20for%20merging%20%22go-team%22%20branch), [python-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2381126%5D%20Request%20for%20merging%20%22python-team%22%20branch), and [crypto-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2381141%5D%20Request%20for%20merging%20%22crypto-team%22%20branch) branches.

### Release Timing

There was also release-process discussion on guix-devel in [Next release?](https://lists.gnu.org/archive/html/guix-devel/?q=Next%20release%3F), which stood out as one of the higher-scoring development-list threads in the bundle.

## Development

### Cross-Compilation Fixes

Cross-compilation and Hurd-related work showed up in the forge data. One patch sets the path to `xauth` when cross-compiling OpenSSH: [gnu: openssh: Set path to xauth when cross compiling](https://codeberg.org/guix/guix/pulls/9037). A related issue tracks `_PATH_BSHELL` coming from the build machine in cross glibc: [cross glibc has _PATH_BSHELL from build machine](https://codeberg.org/guix/guix/issues/9034).

### JavaScript Team Activity

The JavaScript team branch had an active backmerge PR: [javascript-team](https://codeberg.org/guix/guix/pulls/9032). The bundle also caught a patch removing remaining uses of the legacy Python build system from several package modules: [gnu: Remove uses of the legacy python-build-system](https://codeberg.org/guix/guix/pulls/9069).

### Guix Home and Downloads

Two user-facing items stood out on Codeberg: a Guix Home idea for using [`guix home container` as a login script](https://codeberg.org/guix/guix/issues/9055), and a proposed nar-download fix for cases involving `/etc/protocols` and `/etc/services`: [guix: *-download: Fix nar download method](https://codeberg.org/guix/guix/pulls/9072).

### Compression and Text Indexing Packages

BZip3 was added in Guix: [Add BZip3](https://codeberg.org/guix/guix/commit/58595b7dec1307da0107bdc17011d7c8ef19aefd), with `ouch` moving toward unbundled bzip3 support: [ouch: Use unbundled bzip3](https://codeberg.org/guix/guix/commit/0ba11b735fd10f9b374a0d1d571e5337f8b9967e). The bundle also caught [gnu: Add Libsais](https://codeberg.org/guix/guix/commit/6ac23f47e2b2ecc85ec99b7dc6c4255c7177e30c).

### Yosys and Kernel Work

The Yosys-related stack moved to 0.66, including [yosys](https://codeberg.org/guix/guix/commit/48ce256d111c9f861256f2e193a4397e64fb4b03) and [abc-yosyshq](https://codeberg.org/guix/guix/commit/5effeb84936e0a84b030f46eb3bf3d655f78dd03). The related Codeberg PR is [Update yosys saga to 0.66](https://codeberg.org/guix/guix/pulls/9061), and kernel updates were also under review in [kernel updates (2026-06-01)](https://codeberg.org/guix/guix/pulls/9058).
