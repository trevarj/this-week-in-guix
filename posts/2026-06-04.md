## Development

### OpenSSH Cross-Compilation Patch Is Proposed

A Codeberg pull request sets the path to `xauth` when cross-compiling OpenSSH in [gnu: openssh: Set path to xauth when cross compiling](https://codeberg.org/guix/guix/pulls/9037). A related issue tracks `_PATH_BSHELL` coming from the build machine in [cross glibc has _PATH_BSHELL from build machine](https://codeberg.org/guix/guix/issues/9034).

### JavaScript Team Backmerge Is Opened

The JavaScript team branch had a backmerge pull request in [javascript-team](https://codeberg.org/guix/guix/pulls/9032). Another pull request removes remaining uses of the legacy Python build system in [gnu: Remove uses of the legacy python-build-system](https://codeberg.org/guix/guix/pulls/9069).

### Guix Home Container Login Script Is Proposed

A Codeberg issue proposes using [`guix home container` as a login script](https://codeberg.org/guix/guix/issues/9055). A separate pull request proposes a nar-download fix involving `/etc/protocols` and `/etc/services` in [guix: *-download: Fix nar download method](https://codeberg.org/guix/guix/pulls/9072).

Selected package changes this week included additions and updates across compression tools, text indexing libraries, hardware tooling, kernels, Firefox, and Signal Desktop.
