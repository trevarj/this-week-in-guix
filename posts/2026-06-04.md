# This Week in Guix: 2026-06-04
<!-- date: 2026-06-04 -->

This draft covers public Guix activity collected from 2026-05-28 through 2026-06-04.

Collection note: package changes are sampled from a capped diff.

## Top Stories

- The biggest development-list thread this week was about whether Guix should remove the Change-Id commit hook/trailer, with 15 messages collected in the thread: [Consensus on removal of Change-Id commit-msg hook/git trailer?](https://lists.gnu.org/archive/html/guix-devel/?q=Consensus%20on%20removal%20of%20Change-Id%20commit-msg%20hook/git%20trailer%3F).
- GCD 008, "Standing up for human crafting", drew sustained discussion across related guix-devel threads. Start with [Submitting GCD 008: "Standing up for human crafting"](https://lists.gnu.org/archive/html/guix-devel/?q=Submitting%20GCD%C2%A0008%3A%20%E2%80%9CStanding%20up%20for%20human%20crafting%E2%80%9D), and note the related follow-up thread [Submitting GCD 008: "Standing up for humancrafting"](https://lists.gnu.org/archive/html/guix-devel/?q=Submitting%20GCD%20008%3A%20%E2%80%9CStanding%20up%20for%20humancrafting%E2%80%9D).
- The AGit workflow discussion was another high-reply guix-devel thread, suggesting contributor workflow is still an active topic after the Codeberg migration: [AGit workflow](https://lists.gnu.org/archive/html/guix-devel/?q=AGit%20workflow).
- A merge request for the `misc-world-rebuild` branch appeared on guix-patches, with 11 messages collected: [[bug#80782] Request for merging "misc-world-rebuild" branch](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2380782%5D%20Request%20for%20merging%20%22misc-world-rebuild%22%20branch).
- There was also release-process discussion on guix-devel: [Next release?](https://lists.gnu.org/archive/html/guix-devel/?q=Next%20release%3F).

## Development

- Several cross-compilation and Hurd-related issues showed up on Codeberg. One patch sets the path to `xauth` when cross-compiling OpenSSH: [gnu: openssh: Set path to xauth when cross compiling](https://codeberg.org/guix/guix/pulls/9037). A related issue tracks `_PATH_BSHELL` coming from the build machine in cross glibc: [cross glibc has _PATH_BSHELL from build machine](https://codeberg.org/guix/guix/issues/9034).
- The JavaScript team branch had an active backmerge PR: [javascript-team](https://codeberg.org/guix/guix/pulls/9032).
- A patch removed remaining uses of the legacy Python build system in several package modules: [gnu: Remove uses of the legacy python-build-system](https://codeberg.org/guix/guix/pulls/9069).
- There was a user-facing Guix Home idea for using `guix home container` as a login script: [`guix home container` as login script](https://codeberg.org/guix/guix/issues/9055).
- A nar-download fix was proposed for cases involving `/etc/protocols` and `/etc/services`: [guix: *-download: Fix nar download method](https://codeberg.org/guix/guix/pulls/9072).
