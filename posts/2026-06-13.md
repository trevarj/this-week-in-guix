# This Week in Guix: 2026-06-13
<!-- date: 2026-06-13 -->

This issue covers Guix development, package updates, and community discussion from 2026-06-06 through 2026-06-13.

## Top Stories

### Guix Foundation Annual General Meeting Set For 14 June

The Guix Foundation announced its Annual General Meeting for Sunday 14 June at 17:00 UTC, with details sent to the Foundation members mailing list and a public reminder posted to [Mastodon](https://mastodon.social/@futurile/116731928299404290). A follow-up confirmed the [video conference details](https://mastodon.social/@futurile/116736777546613264), and a parallel thread on guix-devel discussed logistics in [Guix Foundation AGM (Sunday 14th)](https://lists.gnu.org/archive/html/guix-devel/?q=Guix%20Foundation%20AGM%20%28Sunday%2014th%29).

### GCD 008 "Standing Up For Human Crafting" Submitted

GCD 008 was submitted for discussion on guix-devel in [Submitting GCD 008: "Standing up for human crafting"](https://lists.gnu.org/archive/html/guix-devel/?q=Submitting%20GCD%C2%A0008%3A%20%E2%80%9CStanding%20up%20for%20human%20crafting%E2%80%9D). A related thread examined copyright and the under-15-line rule in [LLMs and clarifications on < 15 lines and copyright](https://lists.gnu.org/archive/html/guix-devel/?q=LLMs%20and%20clarifications%20on%20%3C%2015%20lines%20and%20copyright), with further follow-up in [[GCD 008]: LLM and copyright infringments](https://lists.gnu.org/archive/html/guix-devel/?q=%5BGCD%20008%5D%3A%20LLM%20and%20copyright%20infringments).

### Nix Flakes And Guix Equivalents Blog Post Circulates

A blog post comparing Nix Flakes with their Guix equivalents was shared by Ludovic Courtès on [Mastodon](https://toot.aquilenet.fr/@civodul/116739238432127929) and discussed further in [a cross-post](https://indieweb.social/@jbz/116738517961494056) and on [lemmy.ml](https://lemmy.ml/post/48469448).

### Team Branch Merge Requests Posted

Merge requests were posted for several team branches, including [misc-world-rebuild](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2380782%5D%20Request%20for%20merging%20%22misc-world-rebuild%22%20branch), [crypto-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2381141%5D%20Request%20for%20merging%20%22crypto-team%22%20branch), [python-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2381126%5D%20Request%20for%20merging%20%22python-team%22%20branch), [go-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2381227%5D%20Request%20for%20merging%20%22go-team%22%20branch), and [gnome-team](https://lists.gnu.org/archive/html/guix-patches/?q=%5Bbug%2381230%5D%20Request%20for%20merging%20%22gnome-team%22%20branch).

### Maintenance Automation Without LLMs Discussed

guix-devel had a thread on contributor tooling in [Automating Maintenance Tasks Without LLMs](https://lists.gnu.org/archive/html/guix-devel/?q=Automating%20Maintenance%20Tasks%20Without%20LLMs), alongside a separate discussion of GnuPG's timeline in [GnuPG 2.4.x branch end of life in July](https://lists.gnu.org/archive/html/guix-devel/?q=GnuPG%202.4.x%20branch%20end%20of%20life%20in%20July).

## Development

### python-next Shebang Breakage Reported

A Codeberg issue reports that the presence of `python-next` breaks the `guix shell` shebang for Python scripts in [Presence of `python-next` breaks `guix shell` shebang for Python scripts](https://codeberg.org/guix/guix/issues/9255).

### Bubblewrap Security Update Proposed

A Codeberg pull request updates bubblewrap to 0.11.2 with security fixes in [gnu: bubblewrap: Update to 0.11.2 [security fixes].](https://codeberg.org/guix/guix/pulls/9274).

### libtpms and swtpm Packaging Work In Progress

A work-in-progress pull request adds TPM emulation libraries in [WIP: "gnu: Add libtpms and swtpm"](https://codeberg.org/guix/guix/pulls/9248).

### GHC Aarch64 Build Fixed

A pull request fixes compiling Haskell code with GHC 9.0 on AArch64 in [gnu: ghc-9.0: Fix compiling Haskell code for Aarch64.](https://codeberg.org/guix/guix/pulls/9240), part of a broader series of GHC packaging cleanups.

### Channel Display And Toolchain Transformation Proposals

A pull request improves how channel instances are displayed in [pull: Display channel instances in a nicer way](https://codeberg.org/guix/guix/pulls/9256), and an issue proposes a new transformation option in [Transformation option `--with-toolchain-input`](https://codeberg.org/guix/guix/issues/9250).

## Packages

### Selected Package Updates

Selected package changes this week included additions, updates, and removals across compilers, editors, games, web tooling, and Haskell libraries.

### GCC 16 And Notable Additions

GCC 16.1.0 was added, and `emacs-jeison`, `emacs-lua-mode`, `emacs-magit-difftastic`, and `emacs-surround` were added to Emacs packages; the jeison addition corresponds to [gnu: Add emacs-jeison.](https://codeberg.org/guix/guix/pulls/9233).

### GIMP, SuperTux, And Ansible Updated

GIMP moved to 3.2.0 with the older `gimp-2` and `gimp-3` package variants removed, SuperTux updated to 0.7.0, and Ansible updated from 10.1.0 to 14.0.0.

### Haskell And Cabal Tooling Refreshed

`cabal-install` moved to 3.12.1.0, `hledger-web` to 1.43.2, and a large set of `ghc-*` library packages were added or updated as part of ongoing Haskell packaging work.

### Nonguix Firefox Security Update

Nonguix updated Firefox to 151.0.3 with security fixes in [nongnu: firefox: Update to 151.0.3 [security fixes].](https://gitlab.com/nonguix/nonguix/commit/4ae06fb5cb75f2ca6b0f2f384f41677ae28c069a).
